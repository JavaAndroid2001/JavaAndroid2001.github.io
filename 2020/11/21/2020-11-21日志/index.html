今日练习题

题目：

题目：定义四个学生对象，添加到集合，并遍历

思路：
1. 自定义Student学生类，四部分。
2. 创建一个集合，用来储存学生对象，泛型：<Student0>
3.根据类，创建4个学生对象
4. 将4个学生对象添加到集合中：add
5. 遍历集合：for、size、get
解法：
public static void main(String[] args) {
    // 1. 学生类创建好了



    // 2. 集合student创建完成     为泛型：Student
    ArrayList<Student> student =new ArrayList<>();


    // 3. 四个对象创建
    Student one = new Student("迪丽热巴",20);
    Student tow = new Student("赵丽颖",21);
    Student three = new Student("杨幂",22);
    Student four = new Student("我",19);



    // 4. 将四个对象加入集合当中
    student.add(one);
    student.add(tow);
    student.add(three);
    student.add(four);


    // 5. 遍历对象student
    for (int i = 0; i < student.size(); i++) {
        Student stu = student.get(i);
        System.out.println("姓名：" + stu.getName() + "，年龄："+stu.getAge());
    }




}
