字符串的进阶一




java.lang.String类代表字符串。



API当中说：Java程序中的所有字符串字面值（如“abc”）都作为此类的实例实现。



其实就是说：程序当中所有的双引号字符串，都是String类的对象。（就算没有new，也照样是。）




字符串的特点：

  
1.字符串的内容永不可变。【重点】

  
2.正是因为字符串不可改变，所有字符串是可以共享使用的。

  
3.字符串效果上相当于是char[]字符数组，但是底层原理是byte[]字节数组。



创建字符串的常见3+1种方式：

  
三种构造方法：

  

  • public String()； 创建一个空白字符串，不含有任何内容。

      
  • public String(char[] array)； 根据字符数组的内容，来创建对应的字符串。

      
  • public String(byte[] array)； 根据字符数组的内容，来创建对应的字符串。

      

一种直接创建：

  
String str =”Hello”; // 右边直接用双引号

练习题

题目：
/*
    题目：
        用一个大集合存入20个随机数数字，然后筛选其中的偶数元素，放到小集合中
        ***要求使用自定义的方法实现筛选

    解决思路：
        1.先创建一个大集合来储存，Random的随机数
        2.自定义方法实现偶素筛选
 */
解法：

public static void main(String[] args) {
          //1. 创建大集合     bigAggregate
      ArrayList<Integer> bigAggregate = new ArrayList<>();
      Random r = new Random();

      for (int i = 0; i < 20; i++) {
          //2. add方法来把随机数加进大集合里
          bigAggregate.add(r.nextInt(100)+1);
      }
          //7. 先把bigAggregate传入getBig让它处理数据得到偶数
      ArrayList<Integer> ok = getBig(bigAggregate);
          //8. 用for循环遍历ok内的偶数
      for (int i = 0; i < ok.size(); i++) {
          //  错误写法：System.out.println(ok);    返回的依旧是： [98, 100, 40, 60, 66, 28, 8, 68]
          //要用get.方法
          System.out.print(","+ok.get(i));
          // 返回值正确：,28,88,24,34,58,72,6,10,52,80,58,14
      }

      // System.out.println(getBig(bigAggregate));        这样写得到的结果为：[98, 100, 40, 60, 66, 28, 8, 68]

  }
         //3.       ok       方法名是待会要用来调用的                  参数名是传入的参数要用到的
  public static ArrayList<Integer> getBig (ArrayList<Integer> numbers){
              //4. 创建了一个小的集合来作为接收后返回给ok
      ArrayList<Integer> small = new ArrayList<>();

      for (int i = 0; i < numbers.size(); i++) {
          //5. 用get方法就能得到numbers给的数值
          int x = numbers.get(i);
          if (x % 2==0){
              small.add(x);
          }
      }
          //6. 反回small的值给方法getBig
      return small;
  }
