继承+this+抽象1

继承的三个特点，this，抽象1

继承的三个特点：


特点1：
Java语言是单继承的
一个类的直接父类只能有唯一一个。


class A {}
class B extends A {} //正确
class C {}
class D extends A,C {} //错误






特点2：


Java语言可以多级继承。


我有一个父亲，我父亲还有一个父亲，也就是爷爷。



最高类为 java.lang.Object


class A {}


class B extends A {} //正确


class C extends B {} //正确





特点3:


一个子类的直接父类是唯一的，但是一个父类可以拥有很多个子类。


可以有很多兄弟姐妹，生二胎。



class A {}


calss B extends A {} //正确


class C extends A {} //正确







this:


this关键字：
用法：
1.在本类的成员方法中，访问本类的成员变量。
2.在本类的成员方法中，访问本类的另一个成员方法。
3.在本类的构造方法中，访问本类的另一个构造方法。

在第三种用法当中要注意：
A. this(...)调用也必须是构造方法的第一个句子，唯一一个。
B. super和this两种构造调用，不能同时使用。
代码区：
class1 Fu:
		public class Fu {
    int num = 99;
}

class2 Zi:

    int num = 20;
    public Zi(){

//        this(1,2);//错误写法
    }
    public Zi(int n){
        this(1,2);
    }

    }

抽象的定义格式：

抽象方法：就是加上abstract关键字，然后去掉大括号，直接分号结束。


抽象类：抽象方法所在的类，必须是抽象类才行。在class之前写上abstract即可
